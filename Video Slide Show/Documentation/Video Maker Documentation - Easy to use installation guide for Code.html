Admob Ads

Modifyng Admob ads

For the proper working of admob ads make sure you have the latest version of google play services installed if not do these steps.

  • Click on Tools > Android > SDK Manager
  • CLick on the SDK Tools tab
  • Select Google Play Services checkbox and then click ok.

 

To change the values of banner and interstitial ad units id do these steps.

  • Open the project tab from the sidebar
  • Now migrate to android > res > values > strings.xml
  • Here change the values with admob ids

Package Name

Modifying Package name

Changes in android manifest

  • Open project tab from sidebar.
  • Now migrate to android > manifest > androidManifest.xml
  • The 3rd line is the package name. Change its value to however you like.

 

 

Changes in Build.gradle file

  • Open project tab from sidebar.
  • Now migrate to Gradle Scripts > build.gradle (Module android)
  • Change the value of applicationId in it.

App Name

Modifying App name

 

  • Open the project tab from the sidebar
  • Now migrate to android > res > values > strings.xml
  • Here change the values with id "app_name"

Build APK

To build a signed apk do these steps

  • Slect build from the top menu and then select Generate Signed APK
  • Select the module as android and click next
  • If you already have a keystore file then use it otherwise make a new one here. Make sure to remember the password of this file and keep this file safe as without this file you will not be able to update your app in the future.
  • Now after filling this form click next where you have to select the directory for the apk and click finish.
